共通・お金BASICS & MONEY

ハウスエッジ控除率 / House Edge
カジノが長期的に取る賭け金の割合。低いほどプレイヤー有利で、ゲーム選びの最重要指標。
期待値EV / Expected Value
1回の賭けで平均的に得（損）する金額。マイナスなら長期的に負ける賭け。
還元率ペイアウト率 / RTP
賭けた総額のうち平均的にプレイヤーへ戻る割合。100%−還元率＝ハウスエッジ。
分散バリアンス / Variance
結果のブレの大きさ。期待値がマイナスでも短期で勝てるのは分散のおかげ。
チップChips / Cheques
カジノ内の通貨。現金と交換して使う。テーブル専用の色付きチップもある。
カラーアップColor Up
細かいチップを高額チップにまとめること。退卓時に「Color up, please」と伝える。
ケージCashier / Cage
チップを現金に換金する窓口。
TITOTicket In Ticket Out
スロット等で使うバーコード付きのチケット。現金の代わりに出入金し、キオスクで換金する。
ミニマムベットTable Minimum
そのテーブルの最低賭け金。卓上のプレートに表示される。
プログレッシブProgressive
プレイ額の一部を積み立てて膨らむジャックポット方式。
ハンドペイHand Pay
$1,200以上など高額配当時に、機械でなく係員が手渡しで支払う手続き。税務書類を伴う。
W-2G
米国の高額ギャンブル配当の申告書類。日本居住者は日米租税条約で還付・免除を申請できる場合がある。

テーブル共通TABLE TERMS

ディーラーDealer
ゲームを進行するカジノ側の担当者。テーブルゲームの相手役も務める。
ピットボスPit Boss
複数テーブルを監督する責任者。トラブル対応やコンプ判断を行う。
ベットBet / Wager
賭けること、または賭け金そのもの。
プッシュPush
引き分け。賭け金は返却される。
シューShoe
複数デッキのカードを入れて配る箱。
プロップベットProposition Bet
配当は大きいが控除率が高い「一発狙い」の賭け。多くは不利。

ブラックジャックBLACKJACK

ヒットHit
カードをもう1枚引くこと。テーブルを軽く叩いて意思表示する。
スタンドStand
これ以上引かず勝負すること。手を横に振って示す。
ダブルダウンDouble Down
賭け金を倍にして、カードを1枚だけ引く強気の選択。
スプリットSplit
同じ数字のペアを2つの手に分けて別々に勝負すること。
サレンダーSurrender
勝ち目が薄い時、賭け金の半分を放棄して降りること。
バストBust
合計が22以上になり即負けとなること。
ナチュラル / ブラックジャックNatural / Blackjack
最初の2枚がA＋10点札で合計21。配当1.5倍（3:2）。
インシュランス保険 / Insurance
ディーラーのAに対する保険。期待値はマイナスで、基本は断るのが正解。
ソフトハンド / ハードハンドSoft / Hard Hand
Aを11として数える手がソフト、そうでない手がハード。
アップカード / ホールカードUp card / Hole card
ディーラーの表向きの札がアップカード、裏向きの札がホールカード。
3:2 / 6:5
ブラックジャック成立時の配当。3:2（1.5倍）が正常で、6:5（1.2倍）の卓は不利なので避ける。
S17 / H17
ディーラーがソフト17でスタンドするのがS17（有利）、ヒットするのがH17（やや不利）。
DASDouble After Split
スプリット後にダブルダウンできるルール。プレイヤーに有利。
基本戦略Basic Strategy
自分の手とディーラーのアップカードから決まる、数学的に最適な打ち方。
カードカウンティングCard Counting
場に出た札を記憶し、残りデッキの有利不利を推定する技術。合法だがカジノは拒否できる。
Hi-Lo
最も普及したカウント法。低い札に+1、高い札に−1を割り当てて合算する。
ランニングカウント / トゥルーカウントRunning / True Count
札の合算値がランニングカウント、それを残りデッキ数で割った実効値がトゥルーカウント。
ベットスプレッドBet Spread
有利な時に大きく、不利な時に小さく賭ける、賭け金の振れ幅。カウンティングの収益源。
偏差プレイ / インデックスDeviation / Index
カウントに応じて基本戦略から外れる打ち方と、その分岐点となる数値。
Illustrious 18 / Fab 4
影響の大きい偏差プレイの代表集。18の主要プレイと、4つの偏差サレンダー。
ペネトレーションPenetration
シャッフルまでに使われるカードの割合。深いほどカウンターに有利。
バックオフBack-off
カジノがカウンター等にプレイ中止や退店を丁寧に促すこと。

ルーレットROULETTE

アメリカン / ヨーロピアン / フレンチAmerican / European / French
0と00の38個がアメリカン（控除率5.26%）、0のみ37個がヨーロピアン（2.70%）。フレンチは返還ルール付きでさらに有利。
インサイドベット / アウトサイドベットInside / Outside Bet
数字の枠内に置く高配当の賭けがインサイド、赤黒や偶奇などの低配当がアウトサイド。
ストレートアップStraight Up
数字1点に賭ける方法。配当35倍。
スプリット / ストリート / コーナーSplit / Street / Corner
隣接する2・3・4個の数字にまたがって賭ける方法（配当17・11・8倍）。
ダズン / コラムDozen / Column
12個の数字にまとめて賭ける方法。配当2倍。
カラーチップColor Chip
プレイヤーごとに色分けされたルーレット専用チップ。誰の賭けか区別するため。
ドリーDolly / Marker
当たり番号の上に置かれるマーカー。外れるまで次の賭けはしない。
ラ・パルタージュ / アンプリゾンLa Partage / En Prison
0が出た時に偶数賭けの半額が戻る（または次回持ち越し）フレンチ卓の救済ルール。控除率1.35%まで低下。
ファイブナンバーベットBasket
0・00・1・2・3の5点賭け。控除率7.89%とルーレット最悪の賭け。避ける。

バカラBACCARAT

バンカー / プレイヤー / タイBanker / Player / Tie
3つの賭け先。バンカーが最有利（控除率1.06%）、タイ（引き分け）は控除率14.36%で論外。
ナチュラルNatural
最初の2枚で合計8か9。出た時点で即決着する。
コミッション手数料 / Commission
バンカー勝ちの配当から引かれる5%の手数料。それでもバンカーが最有利。
ミニバカラMini-Baccarat
少額から遊べる小型テーブルのバカラ。進行が速い。
罫線けいせん / Roadmap
過去の勝敗を記録した表。人気だが数学的には次の結果に影響しない。

クラップスCRAPS

シューターShooter
サイコロを振る人。客が交代で務める。
カムアウトロールCome Out Roll
最初の一投。7・11で即勝ち、2・3・12で即負け。
ポイントPoint
カムアウトで4・5・6・8・9・10が出た時に設定される目標の数字。
パスライン / ドントパスPass / Don't Pass
シューターの勝ちに賭けるのがパスライン、負けに賭けるのがドントパス（わずかに有利）。
カム / ドントカムCome / Don't Come
ポイント確定後に、あらためてパス／ドントパス相当の賭けを始める方法。
オッズベットFree Odds
ポイント確定後に追加できる、控除率0%の完全フェアな賭け。上級攻略の核。
3-4-5xオッズ
ポイント4・10で3倍、5・9で4倍、6・8で5倍までオッズを置ける標準ルール。
プレースベットPlace Bet
特定の数字が7より先に出ることに賭ける方法。6・8は許容だが4・10は不利。
フィールド / ハードウェイField / Hardways
テーブル中央の高控除率ベット群。配当は大きいが不利。
セブンアウトSeven Out
ポイントより先に7が出てシューターの番が終わること。
スティックマン / ボックスマンStickman / Boxman
サイコロを差し出す係と、チップを管理する監督役。クラップスは複数人で運営される。
バブルクラップス電子クラップス / Bubble Craps
ドーム内でサイコロが跳ねる電子版。ミニマムが安く初心者の練習向き。

ポーカーPOKER

テキサスホールデムTexas Hold'em
世界標準のポーカー。手札2枚と共有5枚から最強の5枚を作る。
ホールカード / コミュニティカードHole / Community Cards
自分だけの手札2枚がホールカード、全員共有の場札がコミュニティカード。
フロップ / ターン / リバーFlop / Turn / River
共有カードが開かれる段階。3枚→1枚→1枚の順。
プリフロップPreflop
共有カードが開かれる前、手札2枚だけの最初のベット局面。
ブラインドBlind (SB/BB)
毎ハンド強制的に出す賭け金。$1/$3のようにレートで表す。
チェック / コール / レイズ / フォールドCheck / Call / Raise / Fold
パス、同額を出す、上乗せする、降りる、の4つの基本アクション。
ポジションPosition
自分がベットする順番。後に行動できる（ボタンに近い）ほど情報が多く有利。
ポットPot
場に集まった賭け金の総額。勝者が総取りする。
ショーダウンShowdown
最後まで残ったプレイヤーが手札を見せ合い、勝敗を決めること。
レーキテラ銭 / Rake
カジノが各ポットから抜く手数料。低レートほど相対的な負担が重い。
アウツOuts
自分の手を完成させてくれる残りのカードの枚数。
ポットオッズPot Odds
コール額とポットの比率。ドローの完成率がこれを上回ればコールが正当化される。
4-2の法則Rule of 4 and 2
完成率の暗算法。アウツ×4（残り2枚）／×2（残り1枚）でおおよその%が出る。
スーテッド / オフスートSuited / Offsuit
2枚のスートが同じ（フラッシュを狙える）か、違うか。表記は例：AKs。
コネクターConnector
数字が連続する2枚（例：87）。ストレートを狙える投機的な手。
オープンエンド / ガットショットOpen-ended / Gutshot
両側どちらでも完成する両面ストレート待ち（8アウツ）と、中抜けの待ち（4アウツ）。
タイト・アグレッシブTAG
参加する手を絞り（タイト）、参加したら強く攻める（アグレッシブ）勝ちやすい型。
ティルトTilt
頭に血が上って冷静さを失った状態。技術的な優位を一瞬で溶かす。
バイインBuy-in
卓に着く際に持ち込むチップ額。バンクロールはこの何十回分で管理する。
ストリングベットString Bet
チップを複数回に分けて出す反則行為。レイズ額は一度に、または先に宣言する。

ビデオポーカーVIDEO POKER

Jacks or Better
定番機種。ジャック以上のワンペアから配当がつく。略してJoB。
9/6 / フルペイ9/6 / Full Pay
フルハウス9・フラッシュ6配当の最良の配当表。JoBなら還元率99.54%。
MAX BETMax Bet
5枚賭け。ロイヤルの配当が跳ね上がるため常に選ぶのが鉄則。
ホールド / ドローHold / Draw
残すカードを選ぶのがホールド、残り札を交換するのがドロー。
デノミDenomination
1枚あたりの額面（25¢、$1など）。賭け金は枚数でなくデノミで調整する。
Deuces Wild
2が万能札（ワイルド）になる機種。フルペイなら還元率100%超だが希少。

コンプ・資金管理COMPS & BANKROLL

コンプComp / Complimentary
プレイ実績に応じてカジノが提供する無料の食事・部屋・ショーなどの特典。
プレイヤーズカードPlayer's Card
プレイを記録する無料の会員カード。着席時にディーラーへ渡す。コンプの元になる。
セオTheo / Theoretical Loss
理論上の負け額。コンプはこの数値で決まり、勝ち負けの結果とは無関係。
ホストCasino Host
一定額以上プレイする客につく担当者。部屋・食事・ショーの手配を交渉できる。
メーラーMailer / Offer
プレイ履歴に応じて届く無料宿泊やフリープレイのオファー。
バンクロールBankroll
ギャンブルに充てる総資金。ベット額・セッション予算はこれを基準に管理する。
リスク・オブ・ルーインRisk of Ruin
資金を使い果たして退場する確率。ベットを小さく、資金を厚くするほど下がる。
ハンドルHandle
動かした賭け金の総額。ハウスエッジはこの総額に対して効いてくる。
マーチンゲールMartingale
負けるたびに倍賭けする必勝法。テーブル上限と資金の前に必ず破綻する。
