# ラスベガス カジノガイド

はじめてのカジノ向けの遊び方ガイドサイト。スマホ表示に最適化した1ページ構成（依存ライブラリなし・`index.html` のみ）。

## 収録コンテンツ

- 基礎知識（チップの買い方・換金・マナー・ミニマムベット）
- テーブルゲーム：ブラックジャック / ルーレット / バカラ / ポーカー
- その他：クラップス / スロット / ビデオポーカー

各ゲームは「難易度・控除率バッジ → 遊び方ステップ → 配当表 → 攻略 → 注意点」の共通フォーマット。

## 使い方

`index.html` をブラウザで開くだけ。GitHub Pages でそのまま公開できます
（Settings → Pages → Deploy from a branch → `main` / root）。

## 注意

数値（控除率・配当）はラスベガスの標準ルールに基づく参考値です。カジノ・時期により異なります。
