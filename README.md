# ラスベガス カジノガイド

はじめてのカジノ向けの遊び方ガイドサイト。スマホ表示に最適化。依存ライブラリなしの静的HTML（3ページ構成）。

## ページ構成

| ファイル | 内容 |
|---|---|
| `index.html` | **初心者ガイド** — 基礎知識と各ゲームの遊び方 |
| `strategy.html` | **実践攻略ガイド** — 期待値・完全基本戦略・カードカウンティング・資金管理 |
| `glossary.html` | **用語集** — 全ページ共通。カテゴリ別＋インクリメンタル検索 |

3ページは相互にリンクしています。

### 初心者ガイド（index.html）
- 基礎知識（チップの買い方・換金・マナー・ミニマムベット）
- テーブルゲーム：ブラックジャック / ルーレット / バカラ / ポーカー
- その他：クラップス / スロット / ビデオポーカー
- 各ゲームは「難易度・控除率バッジ → 遊び方ステップ → 配当表 → 攻略 → 注意点」の共通フォーマット。

### 実践攻略ガイド（strategy.html）
- 期待値の原則 / ブラックジャック（基本戦略マトリクス・Hi-Loカウンティング）
- ビデオポーカー（9/6完全戦略）/ クラップス（オッズ）/ バカラ・ルーレットの数学
- テキサスホールデム戦略 / 資金管理・コンプ

### 用語集（glossary.html）
- 101語をカテゴリ別（共通・お金 / テーブル / BJ / ルーレット / バカラ / クラップス / ポーカー / VP / コンプ・資金管理）に収録。
- 検索ボックスに打つと即座に絞り込み・ハイライト表示。

## 使い方

`index.html` をブラウザで開くだけ。GitHub Pages でそのまま公開できます
（Settings → Pages → Deploy from a branch → `main` / root）。相互リンクは公開後（またはローカルの同一フォルダ）で動作します。

## 注意

数値（控除率・配当）はラスベガスの標準ルールに基づく参考値です。カジノ・時期により異なります。
